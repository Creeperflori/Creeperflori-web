Datenschutz

Datenschutzerklärung

Die wichtigsten Informationen zur Verarbeitung von Login-, Support- und Hosting-Daten auf einen Blick.

1. Verantwortliche Stelle

Verantwortlich für die Datenverarbeitung auf dieser Website ist Creeperflori. Kontakt: creeper.flori44@gmail.com und Discord: Creeperflori.

2. Hosting

Die Website wird über GitHub Pages ausgeliefert. Beim Aufruf können technisch notwendige Server-Logdaten wie IP-Adresse, Browsertyp und Zugriffszeit verarbeitet werden.

3. Registrierung und Login

Für Community-Konten verwenden wir Google Firebase Authentication. Verarbeitet werden E-Mail-Adresse und Passwort, wobei das Passwort verschlüsselt gespeichert wird.

4. Support-Anfragen

Nachrichten aus dem Support-Formular speichern wir in Firebase Firestore, damit wir Anliegen bearbeiten und beantworten können. Je nach Kategorie können zusätzliche optionale Angaben wie Discord-Name oder Minecraft-Daten erfasst werden.

5. Lokaler Speicher

Das Login-System nutzt technischen lokalen Speicher im Browser, damit der Anmeldestatus erhalten bleibt. Es werden keine Werbe- oder Tracking-Cookies eingesetzt.

6. Rechte

Du kannst jederzeit Auskunft, Berichtigung oder Löschung deiner Daten anfragen. Kontaktiere uns dafür einfach über das Support-Formular oder per E-Mail.
